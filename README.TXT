Implementar o método Heap-Sort, para ordenação de um conjunto de registos (em cada
registo, a chave é um nome com, até 15 caracteres). O programa deverá apresentar os
passos principais do processo de ordenação.

Para resolver este problema decidi optar por uma Min Heap. Estas são normalmente usadas so para ordenar descentemente, mas isto é facilmente resolvido porque a estrategia que usei foi inserir descentemente na array na orden deceresente. Resultando no resultado pretendido! Uma ordenação por nome, alfabetica, ascendente.
